# Soybean Admin Reference

> **Documentation**: https://docs.soybeanjs.cn
> **GitHub**: https://github.com/soybeanjs/soybean-admin
> **Live Demo**: https://naive.soybeanjs.cn/

## Overview

SoybeanAdmin is a clean, elegant, and powerful admin template based on the latest frontend technologies including Vue3, Vite7, TypeScript, Pinia, NaiveUI, and UnoCSS.

## Tech Stack

| Technology | Version | Purpose |
|------------|---------|---------|
| Vue | 3.x | Frontend framework |
| Vite | 7.x | Build tool |
| TypeScript | Latest | Type safety |
| Pinia | Latest | State management |
| NaiveUI | Latest | UI component library |
| UnoCSS | Latest | Atomic CSS engine |

## Key Features

- **Cutting-edge Tech Stack**: Vue3, Vite7, TypeScript, Pinia, UnoCSS
- **Clean Architecture**: pnpm monorepo structure
- **Strict Code Standards**: ESLint, Prettier, simple-git-hooks
- **TypeScript Support**: Strict type checking
- **Rich Theme Config**: Multiple themes with UnoCSS integration
- **Built-in i18n**: Multi-language support
- **Auto File Routing**: Using [Elegant Router](https://github.com/soybeanjs/elegant-router)
- **Flexible Permission**: Static and dynamic route permissions
- **Mobile Responsive**: Adaptive layouts for all devices

```
└── vite.config.ts        # Vite configuration
```

## Requirements

- **Node.js**: >= 20.19.0
- **pnpm**: >= 10.5.0

## Quick Start

```bash
# Clone repository
git clone https://github.com/soybeanjs/soybean-admin.git

# Install dependencies (MUST use pnpm)
pnpm i

# Start development server
pnpm dev

# Build for production
pnpm build
```

## Adding Custom Pages

### 1. Create Page Component
```
src/views/manage/games/index.vue
```

### 2. Route Auto-Generation
Routes are auto-generated by Elegant Router based on file structure.

### 3. Add Menu Entry
Configure in router settings.

### 4. Add i18n Keys
```typescript
// src/locales/langs/en-us.ts
export default {
  page: {
    manage: {
      games: {
        title: 'Game Management'
      }
    }
  }
}
```
